# Security Policy

## Supported Versions

Currently, only the main branch and the latest release are supported with security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them to @moeshawky privately or email [moeshawky@localhost].

You should receive a response within 48 hours. If the issue is confirmed as a vulnerability, we will open a Security Advisory and work on a patch.
